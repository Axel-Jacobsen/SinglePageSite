Why components

Why?

This project is here to provide a backbone for websites that I would like to make. Hopefully with this backbone, creating new websites will be quick and easy.

In the next section 'components', the components that are available for use will be listed and explained.

Here is an example of the template used in full: Robot Competition

It is obvious that there are still minor errors with the website. Any helping hands are much appreciated!

Components

Here is a cute little caption 

pageDiv

is the base div; every new section of the website must have this class. For div-specific styling (such as coloring), you can add another class. For example, this section's classes are `pageDiv componentDiv`.

If you want a hyperlink to a section, use 

<a href=#sectionId>

For example, this section is linked from the intro via 

<a href=#components>
vFlexOnWidth

is the next most important class. It gives a flex structure to your content.

halfWidth

lets you do stuff like this section, where half of it is a picture/caption, the other is text like this.

There are a lot of other idiosyncrasies with the styles here. Take a look at the style sheet at 

styles/scss/main.scss
